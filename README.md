## INSTALLATION


```
cd ChalhoubFrontendExam
npm install
```

## RUN APP

Make sure you are in ChalhoubFrontendExam directory. Otherwise, go to the directory `ChalhoubFrontendExam`.

```
run npm run start
you may access it at http://localhost:8000
```
